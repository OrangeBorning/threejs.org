[name]

此类存储属性的数据（例如顶点位置，面部索引，法线，与[page：BufferGeometry]相关联的颜色，UV和任何自定义属性，允许用于更有效地将数据传递到GPU。有关详细信息和用法示例，请参阅该页面。

数据存储为任意长度的向量（由[page：BufferAttribute.itemSize itemSize]定义）,并且通常在下面概述的方法中，如果传入索引，则这是自动的乘以矢量长度。

构造函数

[name]( [param:TypedArray array], [param:Integer itemSize], [param:Boolean normalized] )

[page:TypedArray array] -- 必须是[link:https://developer.mozilla.org/en/docs/Web/JavaScript/Reference/Global_Objects/TypedArray TypedArray]. 用于实例化buffer.
这个数组应该有 itemSize * numVertices elements，其中numVertices是关联的[Page：BufferGeometry BufferGeometry]中的顶点数。

[page：Integer itemSize] - 应与之关联的数组的值的数量一个特定的顶点。例如，如果这样属性是存储3分量向量（例如位置，法线或颜色），然后itemSize应为3。

[page：Boolean normalized] - （可选）指示缓冲区中的基础数据如何映射到GLSL代码中的值。例如，如果[page：TypedArray array]是。的实例UInt16Array和[page：Boolean normalized]为true，数组中的值为0 - +65535数据将映射到GLSL属性中的0.0f - + 1.0f。 Int16Array（已签名）将映射从-32767 - +32767到-1.0f - + 1.0f。如果[page：Boolean normalized]为false，则为值将被转换为包含精确值的浮点数，即32767变为32767.0f。

属性

[property:TypedArray array]

[page：TypedArray数组]保存缓冲区中存储的数据。

[property:Integer count]

存储[page：BufferAttribute.array array]的长度除以[page：BufferAttribute.itemSize itemSize]。

如果缓冲区存储3分量矢量（例如位置，法线或颜色）， 那么这将计算存储的这种矢量的数量。

[property:Boolean dynamic]

缓冲区是否是动态的。默认值为* false *。
如果为false，则通知GPU可能经常使用缓冲区的内容而不经常更改。 这对应于[link：https：//developer.mozilla.org/en-US/docs/Web/API/WebGLRenderingContext/bufferData gl.STATIC_DRAW]标志。
如果为true，则通知GPU可能经常使用缓冲区的内容并经常更改。 这对应于[link：https：//developer.mozilla.org/en-US/docs/Web/API/WebGLRenderingContext/bufferData gl.DYNAMIC_DRAW]标志。

[property:Boolean isBufferAttribute]

用于检查此类或派生类是否为BufferAttributes。默认值为* true *。

您不应该更改它，因为它在内部用于优化。

[property:Integer itemSize]

存储在[page：BufferAttribute.array数组]中的向量的长度。

[property:String name]

此属性实例的可选名称。默认值为空字符串。

[property:Boolean needsUpdate]

用于指示此属性已更改且应重新发送到GPU的标志。 修改数组的值时，将其设置为true。

将此设置为true也会增加[page：BufferAttribute.version版本]。

[property:Boolean normalized]

指示缓冲区中的基础数据如何映射到GLSL着色器代码中的值。 有关详细信息，请参阅上面的构造函

[property:Function onUploadCallback]

在Renderer将属性数组数据传输到GPU之后执行的回调函数。

[property:Object updateRange]

对象包含:
[page:Integer offset]: 默认值为* 0 *。开始更新的位置。
[page:Integer count]: 默认值为* -1 *，表示不使用更新范围。

这可用于仅更新存储矢量的某些组件（例如，仅更新组件与颜色有关）。

[property:String uuid]

此对象实例的[link：http：//en.wikipedia.org/wiki/Universally_unique_identifier UUID]。这会自动分配，不应编辑。

[property:Integer version]

版本号，每次[page：BufferAttribute.needsUpdate needsUpdate]属性设置为true时递增。
