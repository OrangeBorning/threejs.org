Draw Mode Constants 绘制模式常量

这些是 [page:Mesh.drawMode] 的有效值，并控制一旦发送到GPU，顶点列表如何被插入。

请注意，这些仅适用于 [page:Mesh.geometry] 是 [page:BufferGeometry]时有效。 当 [page:Mesh.geometry] 是 [page:Geometry] 时更改此项将不起作用。

Draw Modes 绘制模式

THREE.TrianglesDrawMode
这是默认值，并且导致每三个连续顶点（v0，v1，v2），（v2，v3，v5），......被解释为单独的三角形。
如果顶点数不是3的倍数，则忽略多余的顶点。
THREE.TriangleStripDrawMode
这将导致一系列三角形连接在一起，由（v0，v1，v2），（v2，v1，v3），（v2，v3，v4），......给出，以便每个后续三角形与前一个三角形共享两个顶点。
THREE.TriangleFanDrawMode
这将产生一系列三角形，每个三角形共享第一个顶点（如风扇），由（v0，v1，v2），（v0，v2，v3），（v0，v3，v4），......

注意： 从 [link:https://en.wikipedia.org/wiki/DirectX#DirectX_10 DirectX10] 开始，不支持此模式。由于Chrome和Firefox在Windows上使用 [link:https://en.wikipedia.org/wiki/ANGLE_(software) ANGLE] 呈现WebGL，因此内部此模式将转换为支持模式，这可能会导致这些浏览器的性能降低。

使用方法

var geometry = new THREE.Geometry(); geometry.vertices.push( new THREE.Vector3( -10, 10, 0 ), new THREE.Vector3( -10, -10, 0 ), new THREE.Vector3( 10, -10, 0 ), ... ); geometry.faces.push( new THREE.Face3( 0, 1, 2 ), ... ); var material = new THREE.MeshBasicMaterial( { color: 0xffff00 } ); var mesh = new THREE.Mesh( geometry, material ); mesh.drawMode = THREE.TrianglesDrawMode; //default scene.add( mesh );
