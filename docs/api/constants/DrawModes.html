绘制模式常量

[page:Mesh.drawMode]的这些有效值，控制发送到GPU时解释顶点列表。

请注意这些仅适用于 [page:Mesh.geometry] 是 [page:BufferGeometry]。 当改变 [page:Mesh.geometry] 为 [page:Geometry] 将无效。

绘制模式

THREE.TrianglesDrawMode
这是默认值，每三个连续顶点产生一次 (v0, v1, v2), (v2, v3, v5), ... 被解释为一个单独当三角形。
如果顶点数不是3的倍数，则忽略多余的顶点。
THREE.TriangleStripDrawMode
这将导致一系列三角形连接成条带，由下式给出 (v0, v1, v2), (v2, v1, v3), (v2, v3, v4), ... 以便每个后续三角形与前一个三角形共享两个顶点。
THREE.TriangleFanDrawMode
这将导致一系列三角形，每个三角形共享第一个顶点（如风扇）， 由下式给出 (v0, v1, v2), (v0, v2, v3), (v0, v3, v4), ...

注意: 当渲染架构为[link:https://en.wikipedia.org/wiki/DirectX#DirectX_10 DirectX10], 不支持此模式。在 Windows操作系统中，Chrome 和 Firefox 渲染 WebGL 使用 [link:https://en.wikipedia.org/wiki/ANGLE_(software) ANGLE]。 在内部，此模式将转换为支持模式，这可能会导致降低浏览器的性能。

使用方法

var geometry = new THREE.Geometry(); geometry.vertices.push( new THREE.Vector3( -10, 10, 0 ), new THREE.Vector3( -10, -10, 0 ), new THREE.Vector3( 10, -10, 0 ), ... ); geometry.faces.push( new THREE.Face3( 0, 1, 2 ), ... ); var material = new THREE.MeshBasicMaterial( { color: 0xffff00 } ); var mesh = new THREE.Mesh( geometry, material ); mesh.drawMode = THREE.TrianglesDrawMode; //default scene.add( mesh );
