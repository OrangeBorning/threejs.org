材料常量

这些常量定义了所有材质类型共有的属性，除了 Texture Combine Operations 外仅适用于 [page:MeshBasicMaterial.combine MeshBasicMaterial], [page:MeshLambertMaterial.combine MeshLambertMaterial] 和 [page:MeshPhongMaterial.combine MeshPhongMaterial].

面

THREE.FrontSide THREE.BackSide THREE.DoubleSide
定义将要渲染面部的哪一面 - 正面，背面或两者。 默认为 [page:Constant FrontSide].

颜色

THREE.NoColors THREE.FaceColors THREE.VertexColors
[page:Constant NoColors] 是默认值，并将材质的颜色应用于所有面。
[page:Constant FaceColors] 根据每个 [page:Face3 Face3][property:Color color] 的值对面进行着色。
[page:Constant VertexColors] 根据每个 [page:Face3 Face3][property:Array vertexColors] 的值对面进行着色. 这是一个由三个[page：Color Color]组成的数组，一个用于面部中的每个顶点。
更多详细请查看此示例[example:webgl_geometry_colors geometry / colors]。

混合模式

THREE.NoBlending THREE.NormalBlending THREE.AdditiveBlending THREE.SubtractiveBlending THREE.MultiplyBlending THREE.CustomBlending
它们控制发送到 WebGLRenderer 以供 WebGL 使用的材质RGB和Alpha的源和目标混合方程。
[page:Constant NormalBlending] 为默认。
请注意，[page:Constant CustomBlending] 必须设置使用 [page:CustomBlendingEquation Custom Blending Equations].
更多详细请查看此示例[example:webgl_materials_blending materials / blending]

深度模式

THREE.NeverDepth THREE.AlwaysDepth THREE.LessDepth THREE.LessEqualDepth THREE.GreaterEqualDepth THREE.GreaterDepth THREE.NotEqualDepth
材料使用哪种深度函数来比较输入像素Z-深度与当前Z-深度缓冲器值。如果比较结果为真，则将绘制像素。
[page:Materials NeverDepth] 永远不会返回true。
[page:Materials AlwaysDepth] 永远都会返回true。
[page:Materials LessDepth] 如果输入像素Z-深度小于当前缓冲区Z-深度，则返回true。
[page:Materials LessEqualDepth] 是默认值，如果传入像素Z-depth小于或等于当前缓冲区Z-depth，则返回true。
[page:Materials GreaterEqualDepth] 如果输入像素Z-深度大于或等于当前缓冲区Z-深度，则返回true。
[page:Materials GreaterDepth] 如果输入像素Z-深度大于当前缓冲区Z-深度，则返回true。
[page:Materials NotEqualDepth] 如果输入像素Z-深度不等于当前缓冲区Z-深度，则返回true。

纹理组合操作

THREE.MultiplyOperation THREE.MixOperation THREE.AddOperation 这些定义了表面颜色的结果如何与环境贴图（如果存在）相结合，对于[page：MeshBasicMaterial.combine MeshBasicMaterial]，[page：MeshLambertMaterial.combine MeshLambertMaterial]和[page：MeshPhongMaterial.combine MeshPhongMaterial]。
[page:Constant MultiplyOperation] 是默认值，并将环境贴图颜色与表面颜色相乘。
[page:Constant MixOperation] 使用反射率在两种颜色之间进行混合。
[page:Constant AddOperation] 添加两种颜色。
