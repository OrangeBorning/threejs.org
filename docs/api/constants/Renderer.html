WebGLRenderer 常量

面部剔除模式

THREE.CullFaceNone THREE.CullFaceBack THREE.CullFaceFront THREE.CullFaceFrontBack
[page:constant CullFaceNone] 禁用面部剔除。
[page:constant CullFaceBack] 剔除背面（默认）
[page:constant CullFaceFront] 剔除正面。
[page:constant CullFaceFrontBack] 剔除正面和背面。

正面方向

THREE.FrontFaceDirectionCW THREE.FrontFaceDirectionCCW
[page:constant FrontFaceDirectionCW] 将多边形的缠绕顺序设置为顺时针。
[page:constant FrontFaceDirectionCCW] 将多边形的缠绕顺序设置为逆时针（默认）。

阴影类型

THREE.BasicShadowMap THREE.PCFShadowMap THREE.PCFSoftShadowMap
这些定义了WebGLRenderer的[page：WebGLRenderer.shadowMap.type shadowMap.type]属性。

[page:constant BasicShadowMap] 给出未经过滤的阴影贴图 - 速度最快但质量最低。
[page:constant PCFShadowMap] 使用Percentage-Closer Filtering（PCF）算法过滤阴影贴图（默认）。
[page:constant PCFSoftShadowMap] 使用Percentage-Closer Soft Shadows（PCSS）算法过滤阴影贴图。

色调映射

THREE.NoToneMapping THREE.LinearToneMapping THREE.ReinhardToneMapping THREE.Uncharted2ToneMapping THREE.CineonToneMapping
这些定义了WebGLRenderer的[page：WebGLRenderer.toneMapping toneMapping]属性。 用于近似高动态范围（HDR）的外观，标准计算机显示器或移动设备屏幕的低动态范围介质。

[page:constant NoToneMapping] 禁用色调映射。
[page:constant LinearToneMapping] 是默认值。

更多详细请查看此示例[example:webgl_tonemapping WebGL / tonemapping].
